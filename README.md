## Overview

FortiGuard **IOC (Indicators of Compromise) Search** is a feature that enables SOC analysts to query Fortinet’s extensive threat intelligence database for information on potential security threats. It allows users to look up IP addresses, domains, URLs, file hashes, and other threat indicators to determine their reputation and association with malicious activity.

### Key Capabilities:
- **Threat Intelligence Integration:** Leverages FortiGuard Labs' continuously updated database to provide real-time insights.
- **Reputation Analysis:** Helps analysts assess whether an indicator is associated with malware, botnets, phishing, or other threats.
- **Automated and Manual Queries:** Can be used within security workflows to enrich alerts and incidents.
- **SOC Efficiency:** Reduces investigation time by providing actionable threat intelligence directly within FortiSOAR.
